Conecta Jovem

A Conecta Jovem é uma OMG que oferece cursos gratuitos de tecnologia para adolecentes, ajundando jovens a desenvolver habilidades digitais e se preparar para o mercado de trabalho.

Cursos Oferecidos

  • HTML
  • CSS
  • JavaScript

Como se Inscrever

  1. Acesse o link de Inscrição.
  2. Preencha o formulário com seus dados.
  3. Aguarde a confirmação da ONG por e-mail.
